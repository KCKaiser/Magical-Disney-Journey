For this project we found a robust Disney dataset with many variables, in order to evaluate our dataset to see important factors for our model, we had to effective analyze our dataset. We used pandas profiling which allowed us to effectively go through our dataset and apply logic to eliminate any variables that were not needed in our model. We tried using a linear regression model but that produced ineffective results. The more accurate model for our dataset was a Support Vector Regression model, which is part of the scikit-learn models. We tuned our parameters for our model using GridSearchCV, which tests a range of parameters (epsilon, gamma, and c) in the model and test and fits the best parameters for our dataset's model
